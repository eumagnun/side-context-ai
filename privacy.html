Política de Privacidade

Transparência e privacidade. Seus dados sob seu total controle.

Esta política rege o SideContextAI, desenvolvido pela Bifrost Solutions. Valorizamos imensamente sua privacidade e a segurança das suas informações.

1. Não Coleta de Dados

A Bifrost Solutions NÃO coleta, armazena, transmite ou compartilha qualquer tipo de dados pessoais, chaves de API ou histórico de conversas em nossos servidores. Toda a persistência é feita localmente no seu navegador através do armazenamento seguro (chrome.storage.local).

2. Chaves de API e Conexões

Os dados de texto da página aberta e as suas perguntas são transmitidos de forma direta e criptografada do seu navegador para a API do provedor de IA ativo escolhido. A Bifrost Solutions não intermedeia esse tráfego e não tem acesso às suas requisições.

3. Políticas das Provedoras de IA

Ao utilizar a extensão, você fará requisições a provedores de IA terceiros (como Google, OpenAI, Anthropic, DeepSeek, Alibaba, etc.). Cada uma dessas empresas possui suas próprias políticas de privacidade e termos de uso para as chaves de API, que devem ser consultados e observados pelo usuário.

4. Google Drive Integration

Se ativada, a integração opcional de exportação usa tokens OAuth2 do Chrome diretamente com a API oficial do Google Drive para upload. Esses tokens de acesso ficam sob gestão exclusiva do navegador e nunca são expostos a nós.
